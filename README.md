Projeto de Conclusão de Módulo: Lógica de Programação I

Este repositório contém o projeto desenvolvido como parte do programa Desenvolva + Ada Tech & Mercado Livre, concluindo o 1 módulo, de Lógica de Programação I.

O objetivo deste projeto é criar uma aplicação que realiza o cálculo de salário, aplicando os fundamentos de lógica de programação. Durante o desenvolvimento, foram utilizadas boas práticas para garantir a clareza e organização do código.
